    鲛皇见到秦斩不闪不避，竟然想要依靠肉身直接挡下这一击，顿时提醒了一声。

    秦斩闻言一惊，直接身融虚空，逃遁而去，并且在极远的地方出现。
    噗呲！
    虚空一阵低沉的响声，那枚泛着幽蓝色光泽的匕首竟然再次出现在他的后心。
    “有点意思！”
    秦斩喃喃自语，身影再次消散。
    对方的手段极其诡异，而且可以在虚空之中自由地穿梭，并且由隐藏自身气息的秘法，因此一时半刻，秦斩的武道天眼都不能追踪到他。
    对方拿着圣兵噬魂匕首，必定是新晋的五气境无疑！
